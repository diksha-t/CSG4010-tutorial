# CSG4010-tutorial
*Resident Raver*
A 2D platformer built using ImpactJS and the Weltmeister level editor.
**Genre:** Action
**Setting:** College dorm infested with raver zombies.
**Objective:** To reach the end of the level.
**Core Loop:** Move -> Jump across platforms -> Fight zombies -> Reach exit

> This project is based on the *"Building a Game"* chapter from the Unit II material provided.

## Tech Stack
**Engine:** ImpactJS
**Level editor:** Weltmeister - built-in map editor
**Language:** JavaScript - ES5 style as per the material 
**Rendering:** HTML5 '<canvas>'
